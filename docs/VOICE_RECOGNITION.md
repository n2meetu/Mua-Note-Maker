# 语音识别功能使用指南

## 功能说明

Mua-Note-Maker 现已支持微信原生语音识别API，让你可以通过语音快速记录做菜步骤。

## 使用方法

### 1. 基本操作
- 进入「记录步骤」页面
- 点击「按住说话」按钮开始语音识别
- 对着手机说出你的做菜步骤
- 松开按钮结束识别
- 识别结果会自动显示在页面上

### 2. 识别模式
- **持续识别**：长按按钮期间会持续识别语音
- **自动标点**：系统会自动添加标点符号
- **中文优化**：针对中文语音进行了优化

### 3. 混合输入
你可以在语音识别后，继续使用手动输入补充或修改内容，两种方式互不影响。

## 技术实现

### 核心API
使用微信小程序原生的 `wx.createRecognizerContext()` API

```javascript
const recognizerManager = wx.createRecognizerContext()

// 配置参数
const options = {
  lang: 'zh_CN',        // 中文识别
  continuous: true,     // 持续识别
  punct: true          // 自动添加标点
}

recognizerManager.start(options)
```

### 权限配置
已在 `app.json` 中配置必要权限：
```json
{
  "requiredPrivateInfos": [
    "record",
    "microphone"
  ]
}
```

**语音输入**：

**识别结果**：
"艾理干净，清洗3遍，焯水。切好食材备用。下锅炒制，加调料。"

## 常见问题

### Q: 识别不准确怎么办？
A: 可以使用手动输入进行修改，或者重新录音

### Q: 支持方言吗？
A: 微信原生API主要支持普通话，方言识别效果可能不佳

### Q: 需要联网吗？
A: 需要，语音识别需要上传音频到微信服务器处理

### Q: 有识别时长限制吗？
A: 建议单次识别不超过60秒，以获得最佳效果
